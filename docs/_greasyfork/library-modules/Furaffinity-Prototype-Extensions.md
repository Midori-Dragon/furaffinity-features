# <img src="https://raw.githubusercontent.com/Midori-Dragon/furaffinity-features/refs/heads/main/assets/icons/icon.svg" height="30"> Furaffinity Prototype Extensions

A collection of prototype extensions for needed Functions for other Modules.

See documentation on [Furaffinity-Prototype-Extensions](https://midori-dragon.github.io/furaffinity-features/#/Furaffinity-Prototype-Extensions/README).

## How to use

- `@require` this script
  <br>

- Use added Functionalities:
  ```javascript
  let someString = "Hello World";
  someString = someString.trimEnd(" World");
  console.log(someString); // logs: "Hello"
  ```
